# Employee Overview Dashboard (Excel Project)

This project is an Employee Overview Dashboard created using Excel to analyze workforce data and generate business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)
- Slicers & Filters

## Key Insights
- Total employees: 4653 across different cities and education levels
- Majority employees belong to Tier 3 compensation group
- Bangalore has the highest workforce distribution
- Attrition and bench status show important workforce trends
- Hiring peaked in 2017 compared to other years

## Skills Learned
- Data visualization in Excel
- Dashboard design
- Data analysis using pivot tables
- Business insight generation
